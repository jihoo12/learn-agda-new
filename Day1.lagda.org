Square : ∀ {ℓ} {A : Set ℓ} {x0 x1 y0 y1 : A} →
           x0 ≡ x1 → y0 ≡ y1 → x0 ≡ y0 → x1 ≡ y1 → Set ℓ
Square p q r s = PathP (λ i → p i ≡ q i) r s

** Girard's Paradox is type theoretic version of Russell's Paradox
*** core concept
- in the early days of type theory, researchers wanted to create a "Universal Type"
- the type contain all other types and including itself like Type : Type.
- girard prove that Universal Type lead logical explosion
** agda use "Levels" for make code reusable across different levels
** what ℓ represents
- ℓ is a parameter of type Level1
- When you see ∀ {ℓ} → ... it means "for any level ℓ."
- Set ℓ refers to universe at level ℓ.
- By including {ℓ} in your Square definition, This means your Square definition can work for any type A
- regardless of which universe level A lives in.
- making the Square type polymorphic
** Universe Polymorphism
- In Agda the type of a normal type (like Bool or Nat) is Set.
- However Set itself must have a type.
- but if Set : Set then it lead logical error Girard's Paradox
- To solve this, Agda use a sequence
- Bool : Set0
- Set0 : Set1
- Set1 : set2
- ...and so on.
